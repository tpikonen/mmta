MMTA(7)
=======

NAME
----
mmta - a minimal mail transfer agent

SYNOPSIS
--------
mmta aims to be a minimal set of programs for implementing local and
outgoing mail delivery and transfer in a unix system. It can not be
used to accept incoming mail.
